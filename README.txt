// O objetivo deste software é desenvolver uma aplicação que ofereça todos os recursos presentes nas mais completas ferramentas de gestão comercial do mercado tendo uma interface de simples compreeensão pelo consumidor final.
  Reconhecemos que os desenvolvedores tem como principal objetivo oferecer uma ferramenta que atenda todas as necessidades, porém, pecam no desenvolvimento de sua interface final.
  Assim como nosso objetivo também é oferecer o máximo de recursos possíveis, temos como objetivo muito importante também, o comprometimento com a compreensão do nosso consumidor final. 


// Nossa aplicação se encontra em desenvolvimento, se você está lendo este arquivo gostaria de lhe parabenizar por fazer parte deste projeto.


// BetaVersion - A versão beta é disponibilizada a todos aqueles que se dispoem a fornecer informações importantes e relevantes para este projeto, como bugs e erros. Em troca disponibilizamos todos os nossos serviços incluindo suporte, correções e alterações no software. O usuário terá acesso a aplicação de forma antecipada e gratuita.
  As beta versions se dividem em diferentes versões, sempre com seu final contendo números pares. Por ex.: v0.0.46.


// FinalRelease - são as versões livres de erros que o usuário poderá adquirir e desfrutar de todos os nossos serviços com o menor número de erros possíveis, diferente do que ocorre nas versões beta.


// V.Modal - Esta aplicação será dividida em módulos e estes serão distribuidos juntamente com a aplicação principal, porém de forma que seja distribuido nada além do que o cliente utiliza. Dependendo do segmento, diferentes módulos serão incluidos à aplicação principal.
